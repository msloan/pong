# Pong Game

A classic Pong game built with HTML, CSS, and JavaScript.

## How to Play

- **Player 1**: Use W/S or Arrow Up/Down keys to move your paddle
- **Player 2**: Use W/S or Arrow Up/Down keys to move your paddle
- First player to reach 5 points wins!

## Hosting Options

### Option 1: GitHub Pages (Recommended - Free & Easy)

1. Create a GitHub account at https://github.com
2. Create a new repository (name it anything, e.g., "pong-game")
3. Upload all files (index.html, style.css, game.js) to the repository
4. Go to Settings â†’ Pages
5. Under "Source", select "main" branch (or "master")
6. Click Save
7. Your game will be live at: `https://yourusername.github.io/repository-name/`

### Option 2: Netlify (Free & Fast)

1. Create an account at https://www.netlify.com
2. Drag and drop the entire folder onto Netlify's dashboard
3. Your site will be live instantly with a random URL
4. You can customize the domain name in settings

### Option 3: Vercel (Free)

1. Create an account at https://vercel.com
2. Install Vercel CLI: `npm i -g vercel`
3. Run `vercel` in this directory
4. Follow the prompts
5. Your site will be deployed!

### Option 4: Surge.sh (Free)

1. Install Surge: `npm install -g surge`
2. Run `surge` in this directory
3. Follow the prompts to create an account
4. Your site will be live!

## Local Development

Simply open `index.html` in your web browser, or use a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server)
npx http-server
```

Then visit `http://localhost:8000`
